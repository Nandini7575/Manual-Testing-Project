# Manual Testing Project – Demo Web Application

## 📌 Project Overview
This project contains manual testing performed on a demo Selenium-based web application. It includes test case design, execution, and defect reporting.

## 🌐 Application Under Test
https://iamsandesh23.github.io/selenium.github.io/

## 📁 Project File
- ManualTestingProject.xlsx

## 📊 Contents of Excel File
The Excel file contains 2 sheets:

### 1. Test Cases Sheet
- Contains 12 test cases
- Covers login functionality, UI elements, alerts, and validation scenarios
- Includes both positive and negative test cases

### 2. Bug Report Sheet
- Contains 1 defect (TC06)
- Issue: No validation message for empty login submission
- Includes steps to reproduce, expected vs actual result, severity, and priority

## 🧪 Testing Types Performed
- Functional Testing
- UI Testing
- Negative Testing

- Microsoft Excel
- Manual Testing Techniques
